# Gemini CLI Skills

A collection of custom skills for the Gemini CLI to automate and enhance software engineering workflows.

## How to Install a Skill

1.  **Clone this repository** (optional if you just want specific skills).
2.  **Package a skill**: Zip the desired skill folder (e.g., `skills/conductor-compliance-audit`) and rename the extension to `.skill`.
3.  **Install**: Move the `.skill` file to your `.gemini/skills/` directory or use the CLI's installation command.

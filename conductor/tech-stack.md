# Tech Stack: Gemini CLI Skills

## Platforms
- **Gemini CLI**: The primary runtime environment for all custom skills.
- **Conductor**: The project management and workflow orchestration layer.

## Languages & Formats
- **Markdown**: Used for `SKILL.md` definitions and `README.md` documentation.
- **JSON**: Standard format for track metadata and system configurations.

## Target Ecosystems
- **C# / .NET**: Specialized architectural and compliance auditing for .NET projects.
- **Script-based Automation**: Governance and validation for automation scripts (e.g., Python, Bash, PowerShell).

## Distribution & Integration
- **Robust File Copying & SemVer Versioning**: Used for skill installation, version tracking, and update management in target projects.
- **Python Post-installation Hooks**: Optional `post_install.py` scripts for automated project integration (e.g., workflow updates).
